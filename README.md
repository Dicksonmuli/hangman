# Hangman
Word game

## Description
A game in java that helps to develop great practices in a language(English in this case).

## Technologies
Install java:
```
jdk
jre
gradle
```

## Test
```
gradle test
```

## Running
navigate to build/classes/main in the cloned project folder and run
```
java App
```
